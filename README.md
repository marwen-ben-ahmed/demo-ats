# DEMO-ATS ==> SF7.3

## Requirements

If you don't use docker already, follow this link for documentation and installation : [Docker website](https://www.docker.com/)

## DEV Mode :

Run docker-compose :

    docker-compose up -d

Add this line in hosts file

    127.0.0.1 demo.ats.lo

Test this DNS in the browser

    http://demo.ats.lo/
